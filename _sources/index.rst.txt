           </div>
           <div class="panel-body">

Awas is an information flow analyzer and visualizer for component
based systems. The Awas tool consists of OSATE plugin, analysis
algorithms and interactive report generator. Awas OSATE plugin
translates an AADL instance model into Awas intermediate
representation(Awas IR). The Awas report generator enables the user
to explore the model through the transitive dependencies. The process
of generating the reports are highly interactive with the help of a
query language.

The Awas OSATE plugin generates a self contained web-page(here onwards
referred as Awas visualizer) provides an interactive environment to
explore and analyze component based systems. The generated Awas
visualizer is capable of displaying and analyzing the model without relying
on OSATE. Thus facilitating the user to refer the model without installing the
modeling environment.

Awas is an `open source <https://github.com/sireum/v3-awas>`__ project released under
the `Simplified BSD <https://github.com/sireum/v3/blob/release/license.md>`__ license.
